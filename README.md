# jQuery a AJAX
## Potřebné technologie:
* Node JS (https://nodejs.org/en/)

## Instalace v konzoli:
* git clone https://github.com/superucitelka/jquery-ajax-form.git
* cd jquery-ajax-form/server
* npm install

## Spuštění serveru (REST API):
* node index.js

## Klientská aplikace:
* jquery-ajax-form/index.html
